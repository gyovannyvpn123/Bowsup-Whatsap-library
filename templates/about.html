About Bocksup

Project Overview

Bocksup is a Python library that replicates the functionality of yowsup for WhatsApp integration. It provides a complete implementation of the WhatsApp protocol with several key improvements:

  • Modern Python (3.8+) compatibility
  • Asynchronous API using asyncio
  • Improved stability and error handling
  • Complete Signal protocol implementation for secure messaging
  • Better documentation and examples

Architecture

Bocksup uses a layered architecture similar to the original yowsup library:

Core Layers:
  • Network Layer - Handles WebSocket connections to WhatsApp servers
  • Protocol Layer - Implements the WhatsApp message protocol
  • Axolotl Layer - Implements the Signal protocol for encryption
  • Interface Layer - Provides a high-level API for applications
Service Modules:
  • Authentication - Handles WhatsApp authentication
  • Messaging - Manages sending and receiving messages
  • Media - Processes media uploads and downloads
  • Groups - Handles WhatsApp group operations
  • Registration - Manages device registration

Current Limitations

The library is currently not fully functional with real WhatsApp servers. It requires further testing and protocol adaptation with real servers.

See the documentation for details on how to help complete the implementation.
