Instalare și Configurare

Pentru a instala biblioteca Bocksup, utilizați următoarea comandă pip:

pip install bocksup

Cerințe de Sistem

  • Python 3.11 sau mai nou
  • pip (pentru instalare)
  • Biblioteci dependente:
Dependență Versiune Descriere
aiohttp ≥ 3.8.0 Pentru comunicare HTTP asincronă
websockets ≥ 10.0 Pentru comunicare WebSocket
pycryptodome ≥ 3.15.0 Pentru criptare și semnare
construct ≥ 2.10.0 Pentru parsarea și serializarea mesajelor

Configurare Inițială

După instalare, este necesară configurarea bibliotecii prin setarea unor parametri de bază:

import bocksup

# Configurarea bibliotecii
bocksup.Settings.set_debug(True)  # Activează logging detaliat
bocksup.Settings.set_connection_timeout(30)  # Timeout pentru conexiuni în secunde
bocksup.Settings.set_retry_count(3)  # Numărul de reîncercări pentru operații eșuate

Arhitectura Bibliotecii

Bocksup folosește o arhitectură modulară bazată pe conceptul de "layers" (straturi), similar cu modelul OSI pentru rețele:

Layer Responsabilitate Module Principale
Interface Layer API de nivel înalt pentru aplicații bocksup.layers.interface
Axolotl Layer Implementarea Signal Protocol bocksup.layers.axolotl
Protocol Layer Serializarea și deserializarea mesajelor bocksup.layers.protocol
Network Layer Comunicarea prin WebSocket și HTTP bocksup.layers.network

Stack Builder

Pentru a utiliza biblioteca, se creează un "stack" care conectează toate straturile necesare:

from bocksup import create_stack

# Crearea unui stack cu toate straturile
stack = create_stack()

# Sau cu straturi specificate
stack = create_stack(
    interface=True,
    axolotl=True,
    protocol=True,
    network=True
)

Diagrama Arhitecturii

Fluxul de date între straturile arhitecturii:

Aplicație ↔ Interface Layer ↔ Axolotl Layer ↔ Protocol Layer ↔ Network Layer ↔ Serverele WhatsApp

Autentificare

Bocksup suportă multiple metode de autentificare pentru conectarea la serverele WhatsApp:

Autentificare cu Număr și Parolă

from bocksup import auth

# Autentificare cu număr de telefon și parolă
authenticator = auth.Authenticator()
result = await authenticator.login("+40712345678", "password123")

if result.success:
    print(f"Autentificare reușită: {result.user_id}")
else:
    print(f"Eroare de autentificare: {result.error_message}")

Obținerea unui Cod de Asociere (Pairing Code)

from bocksup import auth

# Solicitare cod de asociere
result = await auth.request_pairing_code("+40712345678")

if result.success:
    print(f"Codul de asociere: {result.pairing_code}")
    print(f"Valabil pentru: {result.expiration_time} secunde")
else:
    print(f"Eroare la solicitarea codului: {result.error_message}")

Utilizarea Tokenurilor de Sesiune

from bocksup import auth

# Salvarea tokenului după autentificare
token = result.session_token

# Autentificare ulterioară cu token salvat
authenticator = auth.Authenticator()
result = await authenticator.login_with_token(token)

if result.success:
    print("Autentificare cu token reușită")
else:
    print("Tokenul a expirat sau este invalid")
Considerații de Securitate

Este recomandat să stocați în siguranță tokenurile de sesiune și să implementați măsuri de securitate adecvate pentru a proteja datele de autentificare.

Mesagerie

Modulul de mesagerie permite trimiterea și primirea mesajelor prin WhatsApp:

Trimiterea Mesajelor Text

from bocksup import messaging

# Crearea unui client de mesagerie
message_client = messaging.MessageClient(stack)

# Trimiterea unui mesaj text
result = await message_client.send_text(
    to="+40712345678",
    text="Salut! Acesta este un mesaj de test."
)

if result.success:
    print(f"Mesaj trimis cu ID: {result.message_id}")
else:
    print(f"Eroare la trimiterea mesajului: {result.error}")

Primirea Mesajelor

from bocksup import messaging

# Înregistrarea unui handler pentru mesaje primite
@message_client.on_message
async def handle_message(message):
    print(f"Mesaj primit de la {message.sender}:")
    print(f"Text: {message.text}")
    print(f"Timestamp: {message.timestamp}")
    
    # Răspuns automat
    if "salut" in message.text.lower():
        await message_client.send_text(
            to=message.sender,
            text="Salut! Sunt un bot automat."
        )
