Bocksup

O alternativă îmbunătățită la yowsup pentru integrarea WhatsApp cu stabilitate și gestionare a erorilor superioare

Începe Vezi pe GitHub

Ce este Bocksup?

Bocksup este o bibliotecă Python care replică funcționalitatea de integrare WhatsApp a yowsup, dar cu stabilitate îmbunătățită, gestionare mai bună a erorilor și compatibilitate cu Python modern.

Această bibliotecă oferă o implementare completă a protocolului WhatsApp, permițându-vă să construiți aplicații care pot trimite și primi mesaje, gestiona media, participa la chat-uri de grup și multe altele. 

#!/usr/bin/env python3
import bocksup

# Creează un stack cu credențiale
stack = bocksup.create_stack(
    credentials=("număr_telefon", "parolă"),
    encryption_enabled=True
)

# Conectare și autentificare
await stack.connect()
await stack.authenticate()

# Trimite un mesaj
await stack.send_message({
    "type": "text",
    "to": "1234567890",
    "body": "Salut de la Bocksup!"
})

Caracteristici Principale

🔒

Criptare End-to-End

Mesagerie securizată cu protocolul Signal (Axolotl) pentru criptare end-to-end.

💬

Mesagerie Completă

Suport pentru toate tipurile de mesaje, inclusiv text, media, locație și contacte.

👥

Gestionare Grupuri

Creează și gestionează grupuri, adaugă/elimină participanți și modifică setările grupului.

🖼️

Gestionare Media

Încarcă și descarcă imagini, videoclipuri, fișiere audio și documente.

🧱

Arhitectură Stratificată

Arhitectură modulară, stratificată pentru flexibilitate și extensibilitate.

📱

Înregistrare

Înregistrează conturi noi WhatsApp și verifică numere de telefon.
