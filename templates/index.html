Bocksup

O bibliotecă Python modernă pentru integrare WhatsApp

Bocksup oferă o interfață simplă și robustă pentru conectarea la WhatsApp, trimiterea și primirea de mesaje, și mai mult.

Documentație Exemple
🔒
Autentificare sigură

Asociere și autentificare sigură cu serverele WhatsApp folosind coduri de asociere sau QR.

💬
Mesagerie completă

Trimiteți și primiți mesaje text, imagini, documente și alte tipuri de media.

🛠️
API modern

API asincron bazat pe asyncio, compatibil cu Python 3.8+ și biblioteca WebSocket.

Exemplu de cod rapid 
from bocksup import create_client

async def main():
    # Creează un client WhatsApp
    client = create_client("40712345678")
    
    # Conectează la WhatsApp
    await client.connect()
    
    # Trimite un mesaj text
    await client.send_text_message(
        "40712345678", 
        "Salut de la Bocksup!"
    )
    
    # Deconectare
    await client.disconnect()
