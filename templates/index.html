Bocksup

O bibliotecă Python modernă pentru integrarea cu WhatsApp

Bocksup oferă o implementare completă a protocolului WhatsApp, permițând aplicațiilor să trimită și să primească mesaje, să gestioneze conținut media, să participe la grupuri și multe altele.

Explorează documentația Obține cod de asociere
🔐

Autentificare Sigură

Autentificare simplă și sigură cu suport pentru coduri de asociere și autentificare multi-dispozitiv.

💬

Gestionare Mesaje

API simplu și intuitiv pentru trimiterea și primirea mesajelor, inclusiv suport pentru conținut media.

🔄

Conexiune Fiabilă

Management automat al conexiunii, incluzând reconectare, tratarea erorilor și notificări de stare.

Exemplu de utilizare

import asyncio
from bocksup import create_client

async def main():
    # Creează clientul cu numărul de telefon
    client = create_client("40712345678")
    
    # Conectează și autentifică
    await client.connect()
    
    # Trimite un mesaj
    result = await client.send_text_message(
        to="40787654321", 
        text="Salut de la Bocksup!"
    )
    
    print(f"Mesaj trimis cu ID: {result['id']}")
    
    # Deconectare
    await client.disconnect()

if __name__ == "__main__":
    asyncio.run(main())

Caracteristici

  • ✓ Autentificare cu cod de asociere
  • ✓ Trimitere și primire mesaje text
  • ✓ Criptare end-to-end
  • ≈ Suport pentru conținut media (în dezvoltare)
  • ≈ Gestionare grupuri (în dezvoltare)
  • + Compatibilitate deplină cu Python 3.x
